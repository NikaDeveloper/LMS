# Learning Management System (LMS) API
#### Проект представляет собой бэкенд для системы управления обучением (LMS), реализованный на Django Rest Framework.Система включает управление курсами, уроками, платежами,а также сложную систему прав доступа на основе JWT-авторизации и ролей пользователей.

### *Стек технологий:*
* Фреймворк: Django 5.x
* API: Django REST Framework (DRF)
* Авторизация: JSON Web Tokens (JWT) через djangorestframework-simplejwt
* База данных: PostgreSQL / SQLite (зависит от настроек settings.py)
* Фильтрация: django-filter

### Основной функционал
#### *Проект реализует следующие ключевые возможности:*
* Кастомная модель User: Использование email в качестве основного поля для входа (USERNAME_FIELD).
* Авторизация JWT: Защита всех API-эндпоинтов, кроме регистрации и входа.
* Гибкие права доступа: Реализована ролевая модель на основе групп Django (Moderators):
* Модератор: Может просматривать и редактировать любые курсы/уроки, но не может их создавать или удалять.
* Обычный пользователь: Может видеть, редактировать и удалять только свои курсы/уроки.
* Управление профилем: Динамический выбор сериализатора: при просмотре чужого профиля скрываются чувствительные данные (история платежей, фамилия).
* Платежи: API для просмотра и фильтрации истории оплат курсов и уроков.

## Установка и запуск проекта:
1. Клонирование репозитория
```Bash
git clone [https://github.com/NikaDeveloper/LMS]
cd LMS 
```
2. Создание и активация виртуального окружения
```Bash
python -m venv .venv
```
3. Установка зависимостей
```Bash
pip install -r requirements.txt
```
4. Настройка базы данных и миграции
```Bash
python manage.py makemigrations
python manage.py migrate
```
5. Создание суперпользователя

*Используйте email для входа:*
```Bash

python manage.py createsuperuser
```
6. Запуск сервера
```Bash
python manage.py runserver
```
#### *Проект будет доступен по адресу:* http://127.0.0.1:8000/
